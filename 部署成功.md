## 🌐 下一步：启用 GitHub Pages

### 方法 1：使用 GitHub Actions（自动部署）

1. **访问仓库**: https://github.com/badhope/hardware-studying
2. **点击 Actions 标签**
3. **启用工作流**:
   - 如果是首次使用，点击 "I understand my workflows, go ahead and enable them"
   - 等待部署工作流完成（约 2-3 分钟）
4. **配置 Pages**:
   - 进入 Settings → Pages
   - Source 应自动设置为 "GitHub Actions"

### 方法 2：手动配置 Pages

1. **访问**: https://github.com/badhope/hardware-studying/settings/pages
2. **Build and deployment**:
   - Source: 选择 "Deploy from a branch"
   - Branch: 选择 "main" 和 "/ (root)"
   - 点击 "Save"
3. **等待 1-2 分钟**
