硬件术语词典

收录计算机硬件领域专业术语和技术名词，提供准确、易懂的名词解释，帮助您快速理解硬件知识

ALU

Arithmetic Logic Unit
中级

算术逻辑单元，CPU 的核心组件，负责执行所有的算术运算（加减乘除）和逻辑运算（与或非、比较等）。现代 CPU 包含多个 ALU，可并行处理多条指令。

CPU 架构
示例

Intel Core i9-13900K 包含 32 个 ALU，可在一个时钟周期内执行多条整数运算指令。

FPU CUDA Core

API

Application Programming Interface
中级

应用程序编程接口，软件与硬件之间的通信协议。图形 API（如 DirectX、Vulkan、OpenGL）允许游戏和应用程序调用 GPU 的图形处理能力。

GPU 软件
常见 API

DirectX 12（Windows）、Vulkan（跨平台）、OpenGL（跨平台）、Metal（Apple）

DirectX Vulkan Ray Tracing

BIOS

Basic Input/Output System
基础

基本输入输出系统，主板上的固件，负责开机自检（POST）、硬件初始化和引导操作系统。现代主板多使用 UEFI 替代传统 BIOS，提供更友好的界面和更多功能。

主板 固件
功能

硬件检测、启动顺序设置、CPU 超频、内存 XMP 配置、风扇控制等

UEFI XMP Overclocking

Bandwidth

带宽
基础

带宽指数据传输通道在单位时间内可传输的最大数据量，通常以 GB/s 为单位。内存带宽 = 频率 × 位宽 ÷ 8。带宽越高，数据传输速度越快。

内存 性能
计算示例

DDR4-3200 内存：3200MHz × 64bit ÷ 8 = 25.6GB/s（单通道）

DDR Dual Channel

Cache

高速缓存
基础

缓存是位于 CPU 和内存之间的高速存储器，用于临时存储频繁访问的数据。分为 L1、L2、L3 三级，容量越大、延迟越低，性能越好。L1 最快但容量最小，L3 最慢但容量最大。

CPU 性能
典型配置

AMD Ryzen 7 7800X3D：L1 640KB + L2 8MB + L3 96MB（3D V-Cache）

L1 Cache L3 Cache 3D V-Cache

CUDA

Compute Unified Device Architecture
中级

CUDA 是 NVIDIA 推出的并行计算平台和编程模型，允许软件利用 GPU 的强大计算能力进行通用计算（GPGPU）。广泛应用于科学计算、AI 训练、视频编码等领域。

GPU 计算
应用

AI 深度学习、视频渲染（CUDA 加速）、科学模拟、密码学计算

CUDA Core Tensor Core

Core

核心
基础

核心是 CPU 的实际计算单元，包含完整的指令执行流水线。多核 CPU 可同时执行多个任务，提升多任务处理能力。现代 CPU 通常有 4-16 个物理核心。

CPU 架构
核心数量

Intel Core i5-13600K：14 核心（6P+8E）20 线程
AMD Ryzen 9 7950X：16 核心 32 线程

Thread Hyper-Threading

DDR

Double Data Rate
基础

双倍速率同步动态随机存取存储器，在每个时钟周期的上升沿和下降沿各传输一次数据，带宽是传统 SDRAM 的两倍。已发展至 DDR5 代，频率和带宽持续提升。

内存 标准
DDR 代数

DDR（2000）→ DDR2（2004）→ DDR3（2007）→ DDR4（2014）→ DDR5（2020）

DDR4 DDR5 Bandwidth

DirectX

DX
基础

微软开发的多媒体编程接口集合，包含图形（Direct3D）、声音、输入等 API。DirectX 12 是最新版本，支持光线追踪、可变速率着色等先进特性，仅适用于 Windows。

GPU API
DX12 特性

异步计算、光线追踪（DXR）、可变速率着色（VRS）、网格着色器

API Ray Tracing

DLSS

Deep Learning Super Sampling
中级

深度学习超级采样，NVIDIA RTX 显卡独占技术。利用 AI 和 Tensor Core，将低分辨率画面智能放大到高分辨率，在几乎不损失画质的情况下大幅提升帧率。DLSS 3.0 还支持帧生成技术。

GPU AI
性能提升

DLSS 质量模式：2K 渲染→4K 输出，帧率提升 50-100%
DLSS 3.0 帧生成：帧率可翻倍

Tensor Core RTX FSR

E-Core

Efficiency Core
中级

能效核心，Intel 第 12 代酷睿引入的混合架构设计。E-Core 专注于低功耗处理后台任务和轻量级负载，与高性能的 P-Core 协同工作，提升能效比。

CPU 架构
混合架构

Intel i9-13900K：8 个 P-Core + 16 个 E-Core = 24 核心 32 线程

P-Core Hybrid Architecture

FPU

Floating Point Unit
中级

浮点运算单元，专门负责浮点数（小数）运算的硬件单元。现代 CPU 的 FPU 支持 SIMD 指令集（如 AVX、SSE），可并行处理多个浮点运算，大幅提升科学计算和图形处理能力。

CPU 架构
ALU AVX SIMD

FSR

FidelityFX Super Resolution
中级

AMD 开发的开源超采样技术，通过算法将低分辨率画面放大到高分辨率，提升游戏帧率。FSR 3.0 支持帧生成技术，兼容 NVIDIA 和 Intel 显卡，适用范围广。

GPU 开源
优势

开源免费、兼容所有显卡、无需 AI 硬件、性能提升明显

DLSS XeSS

GPU

Graphics Processing Unit
基础

图形处理器，专门负责图形和图像处理的微处理器。拥有数千个流处理器，擅长并行计算。除图形渲染外，还用于 AI 计算、科学计算、加密货币挖矿等领域。

GPU 基础
主要厂商

NVIDIA（GeForce）、AMD（Radeon）、Intel（Arc）

CUDA Ray Tracing

Hyper-Threading

超线程技术
中级

Intel 的同步多线程技术，允许一个物理核心模拟出两个逻辑核心，同时处理两条线程。通过充分利用 CPU 资源，提升多任务性能约 15-30%。AMD 的类似技术称为 SMT。

CPU 多线程
示例

Intel i7-13700K：8P+8E = 16 核心，支持超线程 = 24 线程（P 核支持，E 核不支持）

SMT Thread

IPC

Instructions Per Clock
高级

每时钟周期指令数，衡量 CPU 架构效率的关键指标。IPC 越高，同频率下性能越强。Zen 3 架构 IPC 提升 19%，Zen 4 再提升 13%，是 AMD 性能反超的关键。

CPU 性能
IPC 提升

AMD Zen 2 → Zen 3：IPC +19%
Intel 11 代→12 代：IPC +18%

Architecture Clock Speed

Latency

延迟
中级

延迟指数据从请求到传输完成所需的时间，通常以纳秒（ns）为单位。内存延迟 = 时序 CL 值 ÷ 频率 × 2000。低延迟对游戏性能影响显著。

内存 性能
计算示例

DDR4-3600 CL16：16 ÷ 3600 × 2000 = 8.89ns
DDR5-6000 CL30：30 ÷ 6000 × 2000 = 10ns

CL Bandwidth

Node

制程节点
基础

制程节点指芯片制造工艺的精细程度，以纳米（nm）为单位。节点数字越小，晶体管越密集，功耗越低，性能越强。主流已进入 5nm、7nm 时代。

CPU 工艺
主流制程

Intel：Intel 7（10nm Enhanced）
AMD：TSMC 5nm、7nm
NVIDIA：TSMC 4N（定制 5nm）

Transistor TSMC

Overclocking

超频
中级

超频是通过提高 CPU、GPU 或内存的工作频率和电压，使其运行在高于出厂设置的状态，从而获得额外性能。需要良好的散热和稳定的电源支持。

CPU 性能
风险提示

可能导致系统不稳定、硬件损坏、失去保修。建议谨慎操作，逐步测试。

PBO XMP

P-Core

Performance Core
中级

性能核心，Intel 第 12 代酷睿引入的混合架构设计。P-Core 专注于高性能计算，支持超线程，频率高，适合游戏和重负载应用。

CPU 架构
混合架构

Intel i5-13600K：6 个 P-Core（支持超线程）+ 8 个 E-Core

E-Core Hybrid Architecture

PCIe

Peripheral Component Interconnect Express
基础

高速串行计算机扩展总线标准，用于连接显卡、SSD、网卡等设备。PCIe 4.0 带宽 16GT/s，PCIe 5.0 翻倍至 32GT/s。显卡通常使用 x16 通道。

主板 接口
带宽对比

PCIe 3.0 x16：15.75GB/s
PCIe 4.0 x16：31.51GB/s
PCIe 5.0 x16：63.02GB/s

NVMe GPU

Ray Tracing

光线追踪
中级

一种渲染技术，通过模拟光线在虚拟环境中的传播路径，生成逼真的光影效果。RTX 显卡的 RT Core 专门加速光线追踪计算，使实时光追成为可能。

GPU 渲染
应用场景

反射（水面、镜面）、阴影（软阴影）、全局光照（间接照明）

RT Core DLSS

RGB

Red Green Blue
基础

红绿蓝三原色 LED 灯效系统，用于硬件外观装饰。通过软件控制颜色和闪烁模式，打造个性化电脑外观。主流同步协议有 ASUS Aura Sync、MSI Mystic Light 等。

外观 灯效
控制软件

ASUS Aura Sync、MSI Mystic Light、GIGABYTE RGB Fusion、Corsair iCUE

ARGB iCUE

SMT

Simultaneous Multi-Threading
中级

同步多线程技术，AMD 版超线程。允许一个物理核心同时执行两条线程，提升 CPU 资源利用率。Ryzen 全系列支持 SMT，16 核心可实现 32 线程。

CPU 多线程
Hyper-Threading Thread

Tensor Core

张量核心
高级

NVIDIA RTX 显卡专用的 AI 计算单元，擅长矩阵运算。用于加速深度学习、DLSS 超采样、AI 降噪等功能。RTX 40 系第三代 Tensor Core 算力达 1.32 TFLOPS。

GPU AI
应用

DLSS 超采样、AI 降噪、广播背景虚化、光线重建（DLSS 3.5）

DLSS RT Core

Thread

线程
基础

线程是 CPU 调度的最小单位。通过超线程/SMT 技术，一个物理核心可模拟出两个逻辑线程。线程数越多，多任务处理能力越强。

CPU 多线程
核心 vs 线程

AMD Ryzen 9 7950X：16 核心 32 线程
Intel i9-13900K：24 核心（8P+16E）32 线程

Core Hyper-Threading

Transistor

晶体管
基础

晶体管是芯片的基本开关单元，控制电流的通断。晶体管数量越多，芯片功能越强大。现代 CPU/GPU 包含数百亿个晶体管，采用纳米级工艺制造。

CPU 基础
晶体管数量

Intel 4004（1971）：2300 个
NVIDIA RTX 4090（2022）：763 亿个
AMD Ryzen 9 7950X：133 亿个

Node Moore's Law

UEFI

Unified Extensible Firmware Interface
中级

统一可扩展固件接口，替代传统 BIOS 的现代固件标准。提供图形化界面、鼠标支持、网络功能、安全启动等特性。支持大于 2TB 的硬盘启动。

主板 固件
UEFI 优势

图形化界面、支持大容量硬盘、启动更快、安全启动、网络更新

BIOS Secure Boot

VRAM

Video Random Access Memory
基础

显存，显卡专用的图像存储器，用于存储纹理、帧缓冲等图形数据。容量越大，可处理的分辨率和画质越高。主流显卡配备 8-24GB GDDR6/GDDR6X 显存。

GPU 显存
显存类型

GDDR6（主流）、GDDR6X（高端）、HBM2/HBM3（专业卡）

GDDR6 Bandwidth

XMP

Extreme Memory Profile
基础

Intel 制定的内存超频配置文件标准，由内存厂商预设好频率、时序和电压。在 BIOS 中一键开启 XMP，即可让内存运行在标称的高性能状态，无需手动超频。

内存 超频
使用示例

购买 DDR4-3600 内存 → BIOS 开启 XMP Profile 1 → 自动运行在 3600MHz CL18

BIOS Overclocking EXPO
