## 📖 使用指南

### 🎯 学习路径推荐

```
首页 → 新手入门 → 学习路径 → 硬件百科 → 装机模拟 → 知识测验
```

> 💡 **适合**: 完全零基础的小白，循序渐进掌握硬件知识

#### 🎓 进阶提升路线

```
技术文章 → 硬件发展史 → 品牌大全 → 术语词典 → 天梯图 → 配置推荐
```

> 💡 **适合**: 有一定基础，想深入了解硬件的同学

#### 💻 实践应用路线

```
装机模拟器 → 配置推荐 → 天梯图 → 知识测验 → 重复练习
```

> 💡 **适合**: 准备装机，需要实战指导的同学
