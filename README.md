## 🌟 项目简介

**硬件探索者**是一个现代化的硬件知识学习平台，专为计算机硬件初学者和爱好者设计。通过系统化的学习路径、互动模拟、知识测验等多种方式，帮助您从零开始掌握硬件知识。

### ✨ 核心特色

- 📚 **系统化学习路径** - 从基础到进阶，循序渐进
- 🔍 **互动式学习** - 装机模拟器、知识测验等互动功能
- 📊 **数据可视化** - 性能对比图表、天梯图排行
- 🎨 **现代化设计** - 玻璃拟态 UI、流畅动画效果
- 📱 **响应式布局** - 完美适配桌面端和移动端
- 🔖 **学习工具** - 书签、笔记、进度追踪等实用功能

## 🎯 功能模块

### 📖 学习资源
- **首页** - 平台概览和快速导航
- **学习路径** - 系统化的学习路线图
- **新手入门** - 零基础友好入门指南
- **硬件百科** - 详尽的硬件知识库
- **手机硬件** - 移动端硬件专题
- **其他设备** - 笔记本、平板等设备知识

### 🛠️ 互动工具
- **装机模拟器** - 虚拟组装电脑，学习硬件搭配
- **知识测验** - 三级难度测试（初级/中级/高级）
- **成就系统** - 学习进度追踪和成就徽章

### 📊 专业工具
- **硬件发展史** - 1971-2024 年硬件发展时间线
- **品牌大全** - 12+ 硬件品牌详细介绍
- **术语词典** - 30+ 专业术语解释
- **天梯图** - CPU/GPU 性能排行榜
- **配置推荐** - 4 档价位装机配置方案

## 🚀 快速开始

### 在线访问

访问我们的 GitHub Pages 站点：
👉 [https://x1882.github.io/hardware-studying/](https://x1882.github.io/hardware-studying/)

### 本地开发

```bash
# 1. 克隆仓库
git clone https://github.com/X1882/hardware-studying.git
cd hardware-studying

# 2. 直接在浏览器打开
# 使用 Live Server 或直接打开 index.html
```

### 推荐开发环境

- **VS Code** + Live Server 扩展
- 任意现代浏览器（推荐 Chrome/Edge/Firefox）
- Node.js 18+（可选，用于开发工具）

## 📖 使用指南

### 学习路径推荐

#### 🔰 零基础入门
1. 浏览 **首页** 了解平台
2. 学习 **新手入门** 建立基础概念
3. 跟随 **学习路径** 系统学习
4. 使用 **硬件百科** 查阅资料

#### 🎓 进阶提升
1. 阅读 **技术文章** 深入学习
2. 查看 **硬件发展史** 了解演变
3. 研究 **品牌大全** 认识厂商
4. 参考 **术语词典** 理解专业词汇

#### 💻 实践应用
1. 使用 **装机模拟器** 练习搭配
2. 查看 **天梯图** 了解性能
3. 参考 **配置推荐** 实战装机
4. 完成 **知识测验** 检验成果

### 功能使用

#### 🔍 全局搜索
- 点击搜索图标或按 `Ctrl+K`
- 输入关键词查找内容
- 支持标题和内容全文搜索

#### 🔖 书签系统
- 点击右下角书签按钮
- 添加/移除当前页面书签
- 书签自动保存到本地存储

#### 📝 笔记功能
- 点击笔记按钮打开面板
- 记录学习笔记和心得
- 支持按页面分类查看

#### 📊 数据可视化
- 天梯图查看性能排行
- 图表对比硬件参数
- 交互式数据探索
