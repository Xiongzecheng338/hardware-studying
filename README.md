## 📖 项目简介

**HardwareHub** 是一个单页面Web应用，旨在通过直观、交互式的方式，帮助新手快速了解电脑硬件知识。从CPU到显卡，从选购指南到配置推荐，用户可以在充满科技感的界面中轻松掌握硬件核心概念。

不再枯燥地阅读长篇大论，点击、悬停、探索，让学习硬件变得像玩游戏一样有趣！

---

## ✨ 功能特性

### 🎨 视觉设计
- **赛博朋克美学**：深色主题搭配霓虹色调（青蓝、紫罗兰、霓虹绿），营造沉浸式科技氛围。
- **动态背景系统**：多层移动光斑、渐变网格背景，让页面"活"起来。
- **玻璃拟态 UI**：半透明磨砂玻璃效果，现代感十足。
- **丰富的表情符号**：使用 Emoji 和特殊符号替代枯燥图标，增加趣味性。

### 🖱️ 交互体验
- **硬件解剖图**：可视化的机箱内部结构，点击部件即可查看详情。
- **详细参数模态框**：包含性能天梯图、关键参数、品牌推荐、选购建议。
- **时间线式引导**：5步学习路径，清晰直观。
- **响应式设计**：完美适配桌面端与移动端，支持触摸操作。

### 📚 内容模块
- **核心硬件详解**：CPU、GPU、内存、存储、主板、电源、机箱、显示器。
- **性能天梯图**：直观展示不同硬件的性能等级。
- **配置推荐**：从入门办公到发烧旗舰，4套精选配置方案。
- **购买指南**：避坑技巧、渠道推荐、保修知识。

### ⚡ 性能优化
- **纯原生开发**：无框架依赖，加载速度快。
- **GPU 加速动画**：流畅的 CSS3 动画与过渡效果。
- **滚动懒加载**：元素进入视口时才触发动画。

## 🚀 快速开始

### 在线访问
直接在浏览器中打开 `index.html` 文件即可。

### 本地运行
1. 克隆仓库：

   bash
git clone https://github.com/badhope/hardware-hub.git

2. 进入项目目录：
bash
cd hardware-hub

3. 打开 `index.html`：
   - 双击文件，或
   - 使用 VS Code 的 Live Server 插件运行。

## 📁 项目结构

由于是单文件应用，结构清晰简单：

hardware-hub/
│
├── index.html # 主页面文件（包含 HTML、CSS、JS）
│
└── README.md # 项目说明文档


### 代码结构说明

- **`<head>`**: 引入 CDN 资源、定义 Tailwind 配置、自定义 CSS 样式（动画、玻璃效果等）。
- **`<body>`**: 
  - **导航栏**：响应式顶部导航，移动端汉堡菜单。
  - **英雄区**：主标题、统计数据、3D 电脑装饰图。
  - **硬件解剖区**：交互式机箱内部结构图。
  - **硬件详情区**：8 个核心硬件的卡片网格。
  - **新手引导区**：时间线式 5 步学习路径。
  - **配置推荐区**：4 档配置方案卡片。
  - **购买指南区**：购物技巧与避坑建议。
  - **页脚**：版权信息与社交链接。
  - **模态框**：硬件详情弹窗。
- **`<script>`**: 
  - 硬件数据对象 (`hardwareData`)。
  - 模态框控制函数。
  - 滚动动画初始化。
  - 移动端菜单逻辑。

## 🎯 设计理念

### 色彩系统
- **主背景**: `#050810` (深邃黑)
- **强调色**: 
  - 青蓝 `#00d4ff` (CPU、科技感)
  - 紫罗兰 `#a855f7` (GPU、神秘感)
  - 霓虹绿 `#00ff88` (RAM、活力感)
  - 粉红 `#ff0080` (存储、时尚感)

### 排版
- **标题字体**: Orbitron —— 未来感、数字化风格。
- **正文字体**: Inter —— 现代无衬线，易读性强。

### 动效
- **移动光斑**: 20秒循环的缓慢移动，营造氛围。
- **悬停反馈**: 卡片上浮、边框发光、图标放大。
- **滚动揭示**: 元素从下方淡入，增强叙事感。
